# Guía de Contribución

Gracias por tu interés en contribuir a Fintech Network.

## Cómo Contribuir

### Reportar Bugs

Usa el template de [Bug Report](.github/ISSUE_TEMPLATE/bug.md) para reportar problemas.

### Sugerir Funcionalidades

Usa el template de [Feature Request](.github/ISSUE_TEMPLATE/feature.md) para proponer nuevas funcionalidades.

### Pull Requests

1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/AmazingFeature`)
3. Commit tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. Push a la rama (`git push origin feature/AmazingFeature`)
5. Abre un Pull Request

## Estándares de Código

- Sigue las convenciones de código del proyecto
- Añade comentarios cuando sea necesario
- Escribe tests para nuevas funcionalidades
- Asegúrate de que los tests pasen antes de hacer PR

## Estructura de Commits

Usa mensajes de commit descriptivos:

```
feat: añadir autenticación con JWT
fix: corregir validación de tarjeta
docs: actualizar README
test: añadir tests para transacciones
```

## Preguntas

Si tienes preguntas, abre un issue con la etiqueta `question`.
