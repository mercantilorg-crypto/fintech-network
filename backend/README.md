# Backend Service

Este módulo alojará la API principal (Node.js) para gestionar usuarios, tarjetas, transacciones y procesos administrativos. Incluye conexiones a MongoDB, Redis y servicios externos simulados.

## Estructura sugerida
- `src/`: código fuente del servicio
- `tests/`: pruebas unitarias e integraciones simuladas
- `package.json`: dependencias del backend
