# Backend

Este módulo alojará los servicios de API y procesamiento de la plataforma fintech. Se propone utilizar Node.js con NestJS o Express.

## Próximos Pasos
- Configurar proyecto base (`package.json`, dependencias core).
- Implementar autenticación (JWT/SSO, almacenamiento en Redis).
- Diseñar endpoints para emisión y gestión de tarjetas.
- Integrar MongoDB y definir esquemas iniciales.
