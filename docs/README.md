# Documentación

Espacio para documentación funcional, técnica y operativa de la plataforma fintech. Incluirá descripciones de arquitectura, requisitos regulatorios, flujos de usuario y manuales de operación.

## Subcarpetas sugeridas
- `architecture/`: diagramas y descripciones técnicas
- `compliance/`: lineamientos AML/KYC y normativas
- `api/`: documentación de endpoints REST/GraphQL
- `runbooks/`: guías de operación y respuesta a incidentes
