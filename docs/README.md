# Documentación

Espacio para especificaciones funcionales, diagramas de arquitectura, manuales de cumplimiento y guías operativas.

## Contenidos Sugeridos
- Arquitectura de referencia (diagramas de servicios y flujos).
- Modelado de datos y esquema de integración con terceros.
- Guías de cumplimiento KYC/AML y procedimientos de auditoría.
- Manual de despliegue y operaciones (CI/CD, monitoreo, alertas).

## Documentos planificados
- `architecture.md`: descripciones de módulos (`auth`, `cards`, `transactions`) y dependencias (MongoDB, Redis).
- `compliance.md`: checklist AML/KYC y procedimientos de auditoría.
- `runbooks/`: guías de resolución ante incidentes (fraude, indisponibilidad de procesadores, degradación de base de datos).
