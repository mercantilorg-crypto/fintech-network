# Frontend Web

Aplicación web basada en React/Next.js para que usuarios y administradores gestionen tarjetas, transacciones y paneles de control.

## Estructura sugerida
- `app/`: páginas y rutas
- `components/`: componentes compartidos
- `public/`: activos estáticos

## Próximos pasos
1. Configurar proyecto Next.js con TypeScript.
2. Implementar autenticación y panel básico de usuario.
3. Integrar dashboards de administración.
