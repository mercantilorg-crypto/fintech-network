# App Móvil

Aplicación móvil construida con React Native para Android e iOS. Permitirá a los usuarios gestionar tarjetas, realizar pagos P2P, recibir notificaciones y consultar saldos.

## Estructura sugerida
- `android/`: configuración específica Android
- `ios/`: configuración específica iOS
- `src/`: componentes y lógica compartida (por crear)

## Próximos pasos
1. Inicializar proyecto React Native (Expo o CLI nativo).
2. Implementar flujo de autenticación y dashboard móvil.
3. Configurar notificaciones push (Expo o Firebase Cloud Messaging).
